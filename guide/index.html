기본 타이머 사용법

✨ 주요 기능

  • 정확한 시간 설정: 시, 분, 초 단위로 정밀 설정
  • 프리셋 시간: 1분, 5분, 10분 등 자주 사용하는 시간 원클릭 설정
  • 시각적 프로그레스: 원형 진행바로 남은 시간 직관적 확인
  • 알림 시스템: 소리 + 브라우저 알림 + 시각적 효과

📝 단계별 사용법

  1. 시간 설정:
    • 시, 분, 초 입력 필드에 원하는 시간 입력
    • 또는 우측 프리셋 버튼(1분, 5분, 10분 등) 클릭
    • 커스텀 시간은 "커스텀 저장" 버튼으로 프리셋에 추가 가능
  2. 타이머 시작: 큰 "시작" 버튼 클릭 또는 스페이스바 누르기
  3. 일시정지/재시작: "일시정지" 버튼 클릭 또는 스페이스바
  4. 리셋: "리셋" 버튼 클릭 또는 R키
  5. 완료: 타이머 완료 시 알림음과 함께 브라우저 알림 표시

💡 프로 팁

풀스크린 모드(설정에서 활성화)를 사용하면 더 집중된 환경에서 작업할 수 있습니다. 브라우저 탭 제목에도 남은 시간이 표시되어 다른 작업 중에도 시간을 확인할 수 있어요!

뽀모도로 기법 마스터하기

🎯 뽀모도로 기법이란?

이탈리아어로 '토마토'를 뜻하는 뽀모도로는 프란체스코 치릴로가 개발한 시간 관리 기법입니다. 25분 집중 + 5분 휴식을 반복하여 집중력을 극대화합니다.

🎯
25분 집중

한 가지 작업에만 몰입

☕
5분 휴식

완전한 휴식과 재충전

🏖️
긴 휴식

4회 완료 후 15-30분

📋 뽀모도로 실행 가이드

  1. 작업 목록 작성: 오늘 할 일을 구체적으로 나열
  2. 우선순위 설정: 가장 중요한 작업부터 시작
  3. 타이머 시작: 25분 뽀모도로 타이머 시작
  4. 집중 유지: 알림, SNS, 메신저 등 모든 방해 요소 차단
  5. 휴식 시간: 5분간 완전히 쉬거나 가벼운 스트레칭
  6. 반복: 4회 완료 후 15-30분 긴 휴식

📊 통계 활용법

뽀모도로 타이머의 일일 통계 기능을 활용해 자신의 생산성 패턴을 파악하세요. 어떤 시간대에 가장 집중이 잘 되는지, 평균적으로 몇 개의 뽀모도로를 완료하는지 추적할 수 있어요.

멀티 타이머 활용법

최대 6개의 타이머를 동시에 실행하여 복잡한 작업이나 여러 프로젝트를 효율적으로 관리할 수 있습니다.

  • 휴식: 10분

🍳 요리

  • 개별 알림: 타이머마다 다른 알림음 설정 가능
  • 이름 설정: "코딩", "운동", "요리" 등 작업명 라벨링
  • 우선순위: 중요한 타이머를 상단에 배치

요리 타이머 가이드

레시피별로 최적화된 타이머로 완벽한 요리를 만들어보세요. 다양한 요리법에 맞는 추천 시간이 제공됩니다.

🥘 요리별 추천 시간

🍚 밥류
  • 백미: 18-20분
  • 현미: 35-40분
  • 찰밥: 25-30분
🍜 면류
  • 라면: 3-4분
  • 스파게티: 8-12분
  • 우동: 5-7분
🥚 계란요리
  • 반숙: 6-7분
  • 완숙: 10-12분
  • 스크램블: 2-3분

🔥 요리 팁

단계별 알림 기능을 활용해 복잡한 요리도 실패 없이! 예를 들어 "5분 후 야채 추가", "10분 후 조미료 넣기" 같은 메모를 설정해보세요.

운동 타이머 사용법

HIIT, 타바타, 인터벌 트레이닝 등 다양한 운동 루틴을 위한 전문 타이머입니다.

  • 총 시간: 4분

🏃 유산소 인터벌

  • 고강도: 1분

문제 해결

⚠️ 자주 묻는 질문

알림음이 들리지 않아요

1. 브라우저 소리 설정 확인

2. 타이머 설정에서 볼륨 조절

3. 브라우저의 사이트 권한에서 소리 허용 확인

브라우저 알림이 오지 않아요

1. 브라우저 주소창 옆 알림 허용 아이콘 클릭

2. 브라우저 설정 → 개인정보 및 보안 → 사이트 설정 → 알림에서 허용

3. 운영체제의 알림 설정도 확인

타이머가 부정확해요

1. 다른 탭에서 무거운 작업 중이라면 일시 중단

2. 브라우저를 최신 버전으로 업데이트

3. 페이지 새로고침 후 다시 시도

설정이 저장되지 않아요

1. 브라우저의 쿠키 및 사이트 데이터 허용 확인

2. 시크릿/프라이빗 모드가 아닌지 확인

3. 브라우저 저장소가 가득 찬 경우 일부 데이터 삭제
